# NightSkyIllustration — Sleep/Evening Mode

I'll create a serene nighttime scene with a full moon, Milky Way arc, and silhouetted hills.

**PLAN:**
- Layer 1: Deep night sky gradient
- Layer 2: Milky Way arc (curved star field)
- Layer 3: Full moon with craters
- Layer 4: Silhouetted hills
- Layer 5: Accent stars and glow

**ELEMENT BUDGET:** ~18 elements total

```tsx
