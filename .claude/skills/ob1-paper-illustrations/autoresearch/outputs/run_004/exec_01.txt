# MapleTreeIllustration — Autumn Maple with Swing

I'll create a paper cut-out illustration of a maple tree in autumn with a wooden swing hanging from a sturdy branch.

**COMPONENT PLAN:**
- Layers: sky | trunk | main canopy (red/orange leaves) | branch | swing | foreground
- Element budget: ~28 elements total
- Subject-first: maple tree trunk → canopy shapes → swing (all within budget before closing)

```tsx
