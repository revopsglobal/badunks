# Badunks

Split-flap display for the Badunks friend group. Static HTML/CSS/JS site.

- **URL:** badunks.vercel.app
- **Deploy:** Vercel on push to main
- **Stack:** Vanilla HTML, CSS, JS. No framework, no build step.
- **Shared messages table:** badunks_shared_messages (Supabase project yyizocyaehmqrottmnaz)
